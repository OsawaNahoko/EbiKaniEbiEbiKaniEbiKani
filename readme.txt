ゲーム概要
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝
えびとかにを仕分けるシンプルな仕分けゲームです。
三年生後期課題の『Drag＆Drop出来る作品』にて制作した作品です。授業としての要項以外に遊びの表現を入れて制作しました。
えびとかに同色の赤色でそれぞれ上下反転した三角形でデザイン出来て共通点がありながら差が出せるという点でモチーフに選びました。
えびとかにの正誤判定を読み取る箇所が多くありためGetConponentをなるべく使用しない疎結合な設計を意識して制作しました。
間隔の変更も容易になるように組み立てています。

操作方法
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝
 
流れてくるトレーの上にエビかカニをドラック＆ドロップをするとえびとかにを乗っけることが出来ます。
看板の指示に合わせて乗せて行きましょう。成績が良くなると工場長の言葉も良くなる。
 
エビをドラック………エビが持てます。
カニをドラック………カニが持てます。
 
エビをドロップ………エビを置けます。
カニをドロップ………カニを置けます。

終了はGameOverの際に表示されるQuitボタンまたは、windowキーを押して終了させて下さい。
